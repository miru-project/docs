# 开发指南

欢迎参与 Miru 的扩展开发，本文档将帮助你快速了解 Miru 的扩展开发方法和注意事项，让你能够轻松地为 Miru 添加新扩展源

本文档的目标受众是想要参与 Miru 扩展开发的贡献者，你需要具备一定的 Next.js 和 Javascript 的基础知识，以及对 Miru 的基本了解。本文档按照开发流程来划分，每个部分包含以下内容：

 - 开发环境：介绍如何搭建 Miru 的扩展开发环境，克隆源码、运行服务等。
 - 开发方法：介绍如何使用 Miru 的接口来实现扩展开发，包括添加新扩展源。
 - 开发示例：给出一些扩展开发的代码示例和注释，帮助你更直观地理解。
 - 开发注意事项：提供一些扩展开发的注意事项和建议，帮助你避免一些常见的错误和问题。
 - 本文档使用正式的语言和图文并茂的方式，以便于你快速、准确、清晰地掌握 Miru 的扩展开发方法。本文档使用线性的顺序，你需要按照顺序阅读每个部分。

本文档假设你已经安装了 Node.js 和 Git。如果你还没有安装，请参考 [Node.js 官网](https://nodejs.org/) 和 [Git 官网](https://git-scm.com/) 进行安装。
