# 开发环境

## 软件

- [Visual Studio Code](https://code.visualstudio.com/)
- [Node.js](https://nodejs.org/en/)
- [Git](https://git-scm.com/)
- [Go](https://golang.org/) `可选`

import { Callout, Steps } from "nextra-theme-docs";

## 部署仓库

<Steps>
  ### 克隆仓库 
  
    ```bash
        git clone git@github.com:miru-project/repo.git 
    ```
  ### 安装 vscode 扩展
    [LiveServe](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer)

### 启动 LiveServe

    使用 vscode 打开克隆的仓库，点击右下角的 Go Live 按钮，即可启动本地仓库服务器

</Steps>

## 本地部署 Miru 环境 `可选`

<Steps>
    ### 克隆仓库

      ```bash
        git clone git@github.com:miru-project/miru-next.git
      ```

    ### 安装依赖
      ```bash
        pnpm i
      ```
      <Callout>
        如果你没有安装 pnpm，请使用一下命令安装 pnpm
        ```bash
          npm i -g pnpm
        ```
      </Callout>

    ### 启动 dev 环境
        ```bash
            pnpm dev
        ```

</Steps>
