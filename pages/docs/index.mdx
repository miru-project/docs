# Miru

**Miru**是一个支持扩展的视频、漫画、小说播放器，它可以让你随时随地在线享受你喜欢的内容。Miru 支持多种数据源扩展，让你可以自由地添加和管理你喜欢的视频、漫画、小说。Miru 还具有简洁的界面设计和高效的性能，让你的观看体验更加流畅和舒适。
